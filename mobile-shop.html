Mobile Shop

Project details

Overview

"Mobile Shop" is a resale product website where users can log in via email, password, or Google sign-in as buyers or sellers. Buyers browse and purchase unsold items with ease, accessing a personalized dashboard to track orders. Sellers manage their inventory and add products through a dedicated dashboard. An admin panel empowers administrators to delete products, manage user accounts, and address reported items, ensuring platform integrity. The site's responsive design, powered by Bootstrap, ensures seamless browsing on various devices. With a focus on user convenience and security, "Mobile Shop" facilitates smooth transactions and aims to be a trusted marketplace for resale products.
